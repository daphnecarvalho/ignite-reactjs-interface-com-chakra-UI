## Project commands

### Prepare project

```bash
  # Install dependencies
  yarn install
```

### Run project

```bash
  # Run server
  yarn server

  # Starts the development server.
  yarn dev
```

### Build project

```bash
  # Builds the app for production.
  yarn build
```

### Run project in production mode

```bash
  # Runs the built app in production mode.
  yarn start
```
